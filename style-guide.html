This style guide defines the visual language for Bits & Bobs, inspired by the classic desktop operating system aesthetic of the early 2000s. All existing pages (landing page, target game, error pages) should follow these conventions.

Typography

Window Title — Bold 13px

Section Heading — Bold 14px

Body text — Regular 13px. The quick brown fox jumps over the lazy dog.

Caption / Status — 12px gray

Font stack: "Trebuchet MS", Tahoma, Geneva, sans-serif

Buttons

All buttons use the raised 3D border pattern. The default (focused) button has a dotted inner outline. Pressed buttons invert the border shading.

Form Inputs

Inputs use the sunken border style (inverted from buttons) with a white background.
